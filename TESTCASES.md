# Тест-сьют "Создание задачи"

## Тест-кейс 1: Позитивный кейс: Проверка создания задачи с валидными данными

### Цель
Проверить, что пользователь может создать задачу, заполнив все поля валидными данными.

### Предусловия
1. Открыть на компьютере браузер Google Chrome  в режиме "Инкогнито"
2. Открыть главную страницу сайта [по ссылке](https://avito-tech-internship-psi.vercel.app/issues)

### Шаги
1. Убедиться, что открыта главная страница  
2. Открыть окно создания задачи  
3. Заполнить поля валидными данными  
4. Сохранить задачу

### Ожидаемый результат
Задача успешно создаётся и отображается в списке задач

---
