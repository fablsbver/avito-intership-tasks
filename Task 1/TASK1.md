# Задание 1

## High Priority

### Некорректная работа фильтров поиска вакансий

**Описание проблемы:**

При применении фильтров результаты выдачи не соответствуют выбранным параметрам:

- выбран город СПБ, отображаются вакансии из МСК
- выбрано "можно удаленно", но выводятся вакансии для офиса
- в выдаче есть вакансии не по профессии "Аналитик"
- в выдаче есть вакансии, не соответствующие чекбоксу "Руководящая роль"

**Риск:**

- сломана логика поиска
- пользователь получает неверные результаты
- может принять неподходящее решение
- снижает доверие к сервису

---

### Чипсы не соответствуют выбранному направлению "Data Science"

**Описание проблемы:**

- в чипсах указано несколько направлений
- выделено "поддержка пользователей"
- в выдаче:
  - нет такого направления
  - отображаются другие направления
  - которые не выбраны ни в фильтрах, ни в чипсах

**Риск:**

- нарушена связь "фильтр → чипсы → выдача"
- пользователь видит противоречивые данные
- интерфейс вводит в заблуждение

---

### Заголовок "Ничего не нашлось по запросу" расположен посередине выдачи

**Описание проблемы:**

- сообщение об отсутствии результатов отображается внутри списка вакансий

**Риск:**

- пользователь может не понять, что результата нет
- нарушается визуальная логика страницы

---

**Описание:**
