
# Баг-репорт: Задачи не сортируются по статусу "Done"

## Описание
При попытке отсортировать задачи по статусу "Done" сортировка не работает, на странице остаются все задачи с разными статусами

**Ожидаемое поведение:**  
На странице со списком задач должны отображаться только задачи со статусом "Done" 

---

## Предусловия
- Открыть браузер Google Chrome 127+ в режиме "Инкогнито"  
- Перейти на главную страницу сайта (ссылка)  
- Созданы задачи с разными статусами (Done, InProgress, etc.)

---

## Шаги для воспроизведения
1. Открыть страницу со списком задач
2. Выбрать сортировку по статусу "Done"  

---

## Ожидаемый результат
- На странице со списком задач отображаются только задачи со статусом "Done" 

---

## Фактический результат
- На странице остаются все задачи с разными статусами
- Сортировка по статусу "Done" не применяется

# Баг-репорт: Название задачи после закрытия окна редактирования не обновляется в списке задач 
## Описание
При редактировании задачи и изменении названия оно не сохраняется,  
после сохранения изменения не отражаются, остаётся старое название задачи в списке задач

---

## Предусловия
- Открыть браузер Google Chrome 127+ в режиме "Инкогнито"  
- Перейти на главную страницу сайта (ссылка)  
- Создана хотя бы одна задача

---

## Шаги для воспроизведения
1. Открыть окно редактирования задачи  
2. Изменить поле "Название задачи" на новое значение  
3. Сохранить задачу  
4. Проверить название в списке задач

---

## Ожидаемый результат
- Название задачи успешно обновляется  
- Новое название отображается в списке задач

---

## Фактический результат
- Название задачи остаётся прежним  
- Изменения не сохраняются
